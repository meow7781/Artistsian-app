# 🚀 Project Name

A modern, high-performance web application built with **Next.js 15**, **TypeScript**, and **Geist Font**.

[![Next.js](https://img.shields.io/badge/Next.js-000000?style=for-the-badge&logo=nextdotjs&logoColor=white)](https://nextjs.org/)
[![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=for-the-badge&logo=typescript&logoColor=white)](https://www.typescriptlang.org/)
[![Vercel](https://img.shields.io/badge/Vercel-000000?style=for-the-badge&logo=vercel&logoColor=white)](https://vercel.com/)

---

## ✨ Features

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Standard for modern Next.js apps)
- **Icons**: [Lucide React](https://lucide.dev/) (Optional, but recommended)
- **Typography**: [Geist Sans & Mono](https://vercel.com/font) for a sleek developer-centric look.
- **Optimization**: Automatic font and image optimization via `next/font` and `next/image`.
